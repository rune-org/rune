# REST-API

Diese Referenzseite ist für entwicklerseitige REST-API-Details reserviert.

Für das Benutzer-Onboarding beginne mit [Erste Schritte](/docs/getting-started).

## Aktuelle Empfehlung

Nutze die Rune-App für Benutzer-Workflows, Zugangsdaten, Vorlagen, Ausführungen, Smith und Scryb.

Wenn du gegen die API entwickelst, verwende das Backend-OpenAPI-Schema als einzige Quelle der Wahrheit und vermeide es, generierten Frontend-Client-Code direkt außerhalb der etablierten API-Wrapper zu importieren.
