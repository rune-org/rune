# API REST

Esta página de referencia está reservada para los detalles de la API REST dirigidos a desarrolladores.

Para la incorporación de usuarios, comienza con [Comenzar](/docs/getting-started).

## Orientación actual

Usa la aplicación Rune para flujos de trabajo de usuario, credenciales, plantillas, ejecuciones, Smith y Scryb.

Si estás desarrollando contra la API, usa el esquema OpenAPI del backend como fuente de verdad y evita importar directamente el código cliente del frontend generado fuera de los envoltorios de API establecidos.
