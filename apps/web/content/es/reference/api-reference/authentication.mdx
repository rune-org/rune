# Autenticación

Esta página de referencia está reservada para los detalles de autenticación dirigidos a desarrolladores.

Para la incorporación de usuarios, comienza con [Comenzar](/docs/getting-started).

## Inicio de sesión de usuarios

Los usuarios de Rune inician sesión a través de la aplicación. Dependiendo de la configuración del espacio de trabajo, la autenticación puede incluir acceso basado en contraseña, SAML o servicios conectados por OAuth utilizados por las credenciales.

## Las credenciales son distintas

La autenticación de la aplicación no es lo mismo que las credenciales del flujo de trabajo.

- La autenticación de la aplicación controla quién puede usar Rune.
- Las credenciales del flujo controlan a qué sistemas externos puede acceder un flujo.

Para orientación sobre credenciales dirigida a usuarios, consulta [Credenciales](/docs/guides/credentials).
