# Compartir

Compartir permite que los compañeros vean, editen o ejecuten flujos de trabajo sin copiarlos.

## Compartir un flujo de trabajo

1. Abre **Flujos de trabajo**.
2. Encuentra el flujo de trabajo.
3. Abre la acción de compartir.
4. Elige el compañero y el rol.
5. Guarda el compartido.

## Roles

Rune admite roles de flujo de trabajo a nivel de usuario, como:

- **Visor:** puede ver el flujo de trabajo.
- **Editor:** puede ver, editar y ejecutar el flujo de trabajo.

Usa el rol más reducido que permita al compañero hacer su trabajo.

## Credenciales compartidas

Un flujo compartido puede requerir credenciales.

Si un compañero puede abrir un flujo pero no ejecutarlo, verifica si la credencial requerida está disponible para él.

Las credenciales deben compartirse con más cuidado que los flujos porque representan acceso a sistemas externos.

## Antes de compartir

Pregúntate:

- ¿Debería este compañero ver la lógica del flujo?
- ¿Debería este compañero poder cambiarla?
- ¿El flujo usa datos o credenciales sensibles?
- ¿Las ediciones afectarán a otros usuarios?

## Revocar acceso

Si alguien ya no necesita acceso, elimínalo de la lista de compartidos del flujo.

Si una credencial ya no debería estar disponible, revócala o rótala por separado.
