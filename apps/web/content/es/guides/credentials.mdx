# Gestionar credenciales

Las credenciales permiten que los flujos de trabajo se conecten a APIs y servicios privados sin poner secretos directamente en el grafo del flujo.

## Cuándo necesitas una credencial

Necesitas una credencial cuando un nodo debe demostrar su identidad a otro servicio.

Ejemplos:

- Llamar a una API privada.
- Enviar un correo electrónico autenticado.
- Conectarse a un proveedor OAuth.
- Ejecutar una integración que requiere una cuenta.

Generalmente no necesitas credenciales para APIs públicas como el punto de acceso GitHub Zen del inicio rápido.

## Crear una credencial

1. Abre **Credenciales**.
2. Haz clic en **Añadir credencial**.
3. Elige el tipo de credencial.
4. Ingresa los valores requeridos.
5. Guarda o conecta la credencial.

Rune almacena los valores sensibles de forma segura y permite que los flujos referencien la credencial por nombre.

## Usar una credencial en un flujo de trabajo

1. Abre el flujo de trabajo en el lienzo.
2. Selecciona el nodo que necesita acceso.
3. Elige la credencial en la configuración del nodo.
4. Guarda el flujo de trabajo.
5. Ejecuta una prueba.

## Compartir con cuidado

Las credenciales pueden compartirse para que los compañeros puedan ejecutar flujos que dependan de ellas.

Antes de compartir, confirma:

- Que el compañero debería estar autorizado a usar esa cuenta o clave.
- Que la credencial solo tiene los permisos que el flujo necesita.
- Que eliminar o cambiar la credencial afectará los flujos que la usan.

## Credenciales OAuth

Para credenciales OAuth, Rune te envía al proveedor, apruebas el acceso y Rune almacena los tokens resultantes para los runs del flujo.

Si OAuth falla, verifica:

- Que la cuenta del proveedor tiene permiso para los ámbitos solicitados.
- Que la credencial fue reconectada después de cambios en los ámbitos.
- Que el administrador del espacio de trabajo ha configurado OAuth correctamente para ese proveedor.

## Eliminar credenciales

Antes de eliminar una credencial, Rune verifica si los flujos la referencian.

Si un flujo todavía usa la credencial, eliminarla puede hacer que los runs futuros fallen.
