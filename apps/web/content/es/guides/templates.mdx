# Plantillas

Las plantillas son puntos de partida reutilizables para flujos de trabajo.

Usa una plantilla cuando quieras una estructura funcional más rápido que construir cada nodo desde cero.

## Explorar plantillas

Abre **Plantillas** para buscar o explorar las plantillas de flujo disponibles.

Una plantilla puede incluir:

- Una estructura de flujo sugerida.
- Nodos y conexiones predefinidos.
- Notas sobre qué personalizar.
- Credenciales o servicios necesarios.

## Usar una plantilla

1. Abre **Plantillas**.
2. Elige una plantilla que coincida con tu objetivo.
3. Revisa los detalles.
4. Haz clic en **Usar**.
5. Guarda el nuevo flujo de trabajo.
6. Reemplaza los valores de ejemplo, credenciales y mensajes.
7. Ejecuta una prueba.

## Personalizar antes de confiar

Las plantillas son puntos de partida, no garantías.

Antes de usar una para trabajo real:

- Renombra el flujo de trabajo.
- Confirma que cada nodo hace lo que esperas.
- Añade o actualiza credenciales.
- Ejecuta con datos de prueba seguros.
- Inspecciona la salida de ejecución.

## Cuándo partir de una plantilla

Usa plantillas cuando:

- Reconoces el patrón del flujo.
- La plantilla ahorra tiempo de configuración.
- Quieres aprender de un ejemplo conocido.

Empieza desde cero cuando el flujo sea pequeño o muy específico.
