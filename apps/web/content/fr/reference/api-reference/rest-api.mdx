# API REST

Cette page de référence est réservée aux détails de l'API REST destinés aux développeurs.

Pour l'intégration des utilisateurs, commencez par [Démarrer](/docs/getting-started).

## Conseils actuels

Utilisez l'application Rune pour les workflows utilisateur, les identifiants, les modèles, les exécutions, Smith et Scryb.

Si vous développez avec l'API, utilisez le schéma OpenAPI du backend comme source de vérité et évitez d'importer directement le code client frontend généré en dehors des wrappers d'API établis.
