# Templates

Templates are reusable workflow starting points.

Use a template when you want a working shape faster than building every node from scratch.

## Browse templates

Open **Templates** to search or browse available workflow templates.

A template may include:

- A suggested workflow structure.
- Prebuilt nodes and connections.
- Notes about what to customize.
- Required credentials or services.

## Use a template

1. Open **Templates**.
2. Choose a template that matches your goal.
3. Review the details.
4. Click **Use**.
5. Save the new workflow.
6. Replace placeholder values, credentials, and messages.
7. Run a test execution.

## Customize before relying on it

Templates are starting points, not guarantees.

Before using one for real work:

- Rename the workflow.
- Confirm every node does what you expect.
- Add or update credentials.
- Run with safe test data.
- Inspect the execution output.

## When to start from a template

Use templates when:

- You recognize the workflow pattern.
- The template saves setup time.
- You want to learn from a known example.

Start from scratch when the workflow is small or very specific.
