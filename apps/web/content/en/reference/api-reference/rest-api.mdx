# REST API

This reference page is reserved for developer-facing REST API details.

For user onboarding, start with [Getting Started](/docs/getting-started).

## Current guidance

Use the Rune app for user workflows, credentials, templates, executions, Smith, and Scryb.

If you are developing against the API, use the backend OpenAPI schema as the source of truth and avoid importing generated frontend client code directly outside the established API wrappers.
