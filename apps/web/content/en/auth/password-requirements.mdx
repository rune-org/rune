# Password Requirements

All passwords in Rune must meet these security requirements.

## Requirements

Your password must have:

- At least 8 characters
- One uppercase letter (A-Z)
- One lowercase letter (a-z)
- One number (0-9)
- One special character: `! @ # $ % ^ & * ( ) , . ? " : { } | < >`

## Examples

**Strong passwords:**

- `TruckBlue$89`
- `Coffee!Morning42`
- `Hy9$PerSecure!`

**Weak passwords that won't work:**

- `password` - too common
- `12345678` - no letters or special characters
- `Password` - missing number and special character
- `Pass123` - too short

## Tips for Creating Strong Passwords

Use a memorable phrase with numbers and special characters. For example, "I drink 3 cups of coffee!" could become `IDrink3Cups!`.

## Where These Apply

Password requirements apply when:

- Creating your admin account
- Changing your password
- An admin resets your password
