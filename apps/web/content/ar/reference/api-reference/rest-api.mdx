# واجهة برمجة التطبيقات REST

هذه الصفحة المرجعية محجوزة لتفاصيل واجهة برمجة التطبيقات REST الموجهة للمطورين.

لإعداد المستخدمين، ابدأ بـ[البدء](/docs/getting-started).

## التوجيه الحالي

استخدم تطبيق Rune لسير عمل المستخدم وبيانات الاعتماد والقوالب وعمليات التنفيذ وSmith وScryb.

إذا كنت تطوّر ضد واجهة برمجة التطبيقات، استخدم مخطط OpenAPI للواجهة الخلفية كمصدر الحقيقة الوحيد، وتجنب استيراد كود العميل الأمامي المُولَّد مباشرةً خارج مغلّفات واجهة برمجة التطبيقات المُنشأة.
