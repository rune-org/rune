# المصادقة

هذه الصفحة المرجعية محجوزة لتفاصيل المصادقة الموجهة للمطورين.

لإعداد المستخدمين، ابدأ بـ[البدء](/docs/getting-started).

## تسجيل دخول المستخدمين

يسجّل مستخدمو Rune الدخول عبر التطبيق. بناءً على إعداد مساحة العمل، قد تتضمن المصادقة الوصول المستند إلى كلمة المرور أو SAML أو الخدمات المتصلة بـOAuth المستخدمة من قِبل بيانات الاعتماد.

## بيانات الاعتماد منفصلة

مصادقة التطبيق ليست هي نفسها بيانات اعتماد سير العمل.

- تتحكم مصادقة التطبيق في من يمكنه استخدام Rune.
- تتحكم بيانات اعتماد سير العمل في الأنظمة الخارجية التي يمكن لسير العمل الوصول إليها.

للحصول على إرشادات بيانات الاعتماد الموجهة للمستخدمين، راجع [بيانات الاعتماد](/docs/guides/credentials).
