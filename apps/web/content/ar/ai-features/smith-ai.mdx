# منشئ Smith الذكي

يساعدك Smith على تحويل وصف بلغة طبيعية إلى مسودة سير عمل.

استخدم Smith عندما تعرف النتيجة التي تريدها لكنك لا تريد وضع كل عقدة يدوياً.

## البدء مع Smith

1. افتح **إنشاء**.
2. اختر **سؤال وكيل** أو افتح Smith من اللوحة.
3. صف سير العمل الذي تريده.
4. راجع العقد والاتصالات المُولَّدة.
5. احفظ سير العمل.
6. شغّل تنفيذاً تجريبياً.

## كتابة تعليمة مفيدة

التعليمات الجيدة تتضمن:

- الحدث الذي يبدأ سير العمل.
- الخدمات المعنية.
- قواعد القرار.
- الإجراء النهائي.
- أي قيود أو حالات خاصة.

مثال:

```text
Build a workflow that starts from a webhook, checks whether the payload priority is high, summarizes the request with an AI agent, and logs the summary.
```

## المراجعة قبل التشغيل

قد يرتكب Smith أخطاء. قبل الاعتماد على سير عمل مُولَّد:

- تحقق من إعدادات كل عقدة.
- تأكد من اختيار بيانات الاعتماد المطلوبة.
- اقرأ شروط الفرع بعناية.
- شغّل ببيانات اختبار آمنة.
- افحص مخرجات التنفيذ.

## التكرار مع Smith

بعد المسودة الأولى، اطلب تغييرات محددة.

أمثلة:

```text
Add a Log node after the HTTP request so I can see the response body.
```

```text
Change the branch so only urgent support tickets continue to the email step.
```

الطلبات الصغيرة أسهل في المراجعة من طلب Smith لإعادة بناء كل شيء دفعة واحدة.
